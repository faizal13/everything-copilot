# Java Spring Boot Skill

## Name
Java Spring Boot

## Description
Spring Boot application patterns, JPA/Hibernate best practices, REST API design, Spring Security, testing strategies, and production-ready configuration for Java 17+ applications.

## Trigger Conditions
- Working on Java source files (*.java)
- Spring Boot configuration (application.yml, application.properties)
- Maven or Gradle build files (pom.xml, build.gradle)
- JPA entities, repositories, or database migrations
- Spring Security configuration
- REST controller development
- Files matching: `*.java`, `pom.xml`, `build.gradle*`, `application*.yml`, `application*.properties`

## Files
- `spring-boot-patterns.md` — Application architecture, dependency injection, configuration
- `jpa-hibernate.md` — Entity design, repositories, N+1 prevention, transactions
- `spring-security.md` — Authentication, authorization, CSRF, CORS, JWT
- `spring-testing.md` — JUnit 5, MockMvc, Testcontainers, integration tests

## Model Recommendation
- **Sonnet** for implementation, controller/service/repository code
- **Opus** for architecture decisions, security configuration
- **Haiku** for boilerplate generation, DTO creation
